# Keybindings

LogParser uses vim-style navigation.

## Navigation

| Key | Action |
|-----|--------|
| `j` / `↓` | Move down one line |
| `k` / `↑` | Move up one line |
| `h` / `←` | Scroll left |
| `l` / `→` | Scroll right |
| `gg` | Jump to start of file |
| `G` | Jump to end of file |
| `:N` | Jump to line N |

## Search

| Key | Action |
|-----|--------|
| `/` | Enter search mode |
| `n` | Jump to next match |
| `N` | Jump to previous match |

## General

| Key | Action |
|-----|--------|
| `:q` | Quit |
